## Лабиринт сокровищ

Текстовая игра, в которой игрок исследует лабиринт, избегает ловушек, решает загадки и пытается выбраться с сокровищем.

## Установка

Требуется установленный Poetry и Python 3.12.

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd labyrinth_treasure
make install
```

## Запуск игры

```bash
make project
```
или напрямую через Poetry:

```bash
poetry run labyrinth
```

## Игровой процесс

Вы начинаете у входа в лабиринт и можете перемещаться между комнатами, подбирать и использовать предметы, попадать в ловушки и решать загадки. Цель — найти сокровище и выбраться живым.

### Основные команды

- `help` — показать список команд.
- `look` — осмотреть текущую комнату.
- `go <направление>` — перейти в другую комнату (например, `go north`).
- `inventory` — показать инвентарь.
- `take <предмет>` — взять предмет из комнаты (например, `take key`).
- `use <предмет>` — использовать предмет (например, `use key`, `use treasure`).
- `solve <ответ>` — ответить на загадку в комнате (например, `solve завтра`).
- `quit` — выйти из игры.

Во время игры отображается текущее количество очков здоровья (HP). Попадание в ловушки уменьшает HP, а некоторые случайные события могут его восстановить. При падении HP до нуля игра завершается.

## Случайные события и ловушки

В некоторых комнатах есть ловушки, которые могут нанести урон или телепортировать игрока к входу. В зале иногда срабатывают случайные события: можно найти зелье лечения или столкнуться с внезапным сквозняком.
